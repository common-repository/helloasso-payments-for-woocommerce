== Description ==

L’extension **HelloAsso Payments for WooCommerce** vous permet d’ajouter HelloAsso comme moyen de paiement sur votre boutique WooCommerce.
Vous n'avez pas de compte sur HelloAsso, [créer votre compte en quelques minutes ici.](https://auth.helloasso.com/inscription?from=woocommerce)

Une fois votre compte HelloAsso lié, vos paiements seront réalisés gratuitement (sans frais et sans commission pour votre association) par HelloAsso via le module [HelloAsso Checkout](https://www.helloasso.com/outils/checkout)
Tous les paiements réalisés via votre boutique WooCommerce apparaitront dans votre compte HelloAsso comme des paiements réalisés via le Checkout avec le numéro de commande WooCommerce.

Vous pouvez créer un compte de test sur [notre environnement de test](https://www.helloasso-sandbox.com) afin de tester des paiements sur votre boutique.

L’extension **HelloAsso Payments for WooCommerce** utilise l'API externe d'HelloAsso pour effectuer les paiements et la connexion de votre association.
En utilisant ce plugin, vous acceptez les [conditions générales d'utilisation](https://www.helloasso.com/cgu-associations/2023-10) et à la [politique de confidentialité](https://www.helloasso.com/confidentialite) de HelloAsso.

== Installation ==

1. Téléchargez le plugin sur votre WordPress
2. Activez dans la partie 'Extensions' du back-office WordPress
3. Créez-vous un compte sur helloasso.com ou liez votre compte existant.

== Frequently Asked Questions ==

= Qu'est-ce que HelloAsso ? =

[HelloAsso](https://www.helloasso.com) est une solution gratuite qui permet à toutes les associations françaises de recevoir des paiements sur internet, sans frais ni commission. Plus de 80 000 associations en France font confiance à HelloAsso pour financer leurs activités et gagner du temps.

= Est-ce que HelloAsso est vraiment gratuit ? =
Oui ! [HelloAsso ne prélève aucun frais ni aucune commission](https://www.helloasso.com/comment-ca-marche) sur les paiements qui sont effectués par vos contributeurs. Il n'y a aucune limite d'utilisation, que ce soit en nombre de formulaires ou bien en montant collecté. Souhaitant faire profiter de notre solution à toutes les associations, nous avons choisi un modèle alternatif basé sur la contribution volontaire des internautes. Lorsqu'un internaute effectue un paiement à votre association (par exemple en achetant une place pour votre prochain événement), il a la possibilité de laisser une contribution volontaire facultatif à HelloAsso. Cette contribution volontaire est libre, c'est à dire que l'internaute peut choisir le montant de la contribution volontaire qu'il souhaite donner, ou bien de ne pas laisser de contribution volontaire.

[En savoir plus sur le model économique d'HelloAsso.](https://www.helloasso.com/comment-ca-marche)

= Qui peut utiliser HelloAsso ? =
Seules les associations françaises déclarées et qui possèdent un compte bancaire au nom de l'association peuvent utiliser HelloAsso. Si vous êtes une entreprise ou un particulier, vous ne pouvez pas collecter de l'argent via HelloAsso. En revanche vous pouvez évidemment contribuer à des projets associatifs !

= Comment utiliser l'extension HelloAsso ? =
Ce plugin vous permet de sélectionner HelloAsso comme moyen de paiement pour votre boutique. Ainsi, vous bénéficiez des avantages de la boutique tout en récoltant 100% des montants des paiements (sans frais, sans commission).
